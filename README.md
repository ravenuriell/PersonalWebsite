# Personal Website

This repository hosts a personal portfolio website built with **React**, showcasing personal and school projects, internships, and information about me. The site is optimized for responsive design and smooth transitions.

## Features

- **Responsive Layout**: Adaptable sections for desktop and mobile screens.
- **Landing page**: Website introduction and navigation.  
- **About Section**: Highlights experience and personal background.
- **Projects Section**: Highlights personal and school projects, also github repositories.
- **Contacts Section**: Usable contacts page that has input validation. 

## Platforms used

- **React**  
- **React Icons**  
- **CSS3 with animations**  
- **GitHub Pages** for deployment
